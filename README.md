## Installing
```
$ git clone https://github.com/johnnyyu0930/parcel-react-app.git

$ cd $PROJECT_NAME

$ npm install

or

$ yarn

```

## dev
```
$ npm start 

or 

$ yarn start

open http://localhost:3000
```

## build
```
$ npm run build

or

$ yan run build
```
